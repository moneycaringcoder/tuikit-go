# Roadmap

## v0.12 (In Progress)

- **SSH serving** — serve tuikit-go apps over SSH via Wish integration
- **Cosign signature verification** — verify release binary signatures
- **Delta updates** — binary patch-based self-update for smaller downloads
- **Docs site** — this MkDocs Material site with component gallery
- **Starter template** — `go install` scaffold for new projects
- **Go Report Card A+** — lint and vet clean pass

## Planned

- **Tree component** — collapsible tree view for hierarchical data
- **Form component** — multi-field validated form with keyboard navigation
- **Split pane** — arbitrary n-way splits beyond DualPane
- **Markdown renderer** — render Markdown in a scrollable viewport
- **WASM target** — compile TUI apps for browser via `wasip1`
- **More theme presets** — Solarized, Everforest, Nightfox
