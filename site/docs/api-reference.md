## Key Types Quick Reference

| Type | Description |
|------|-------------|
| `Component` | Interface all widgets implement |
| `Themed` | Interface for theme-aware components |
| `Theme` | Semantic color token struct |
| `Table` | Adaptive data table |
| `ListView[T]` | Generic scrollable list |
| `LogViewer` | Append-only log display |
| `Tabs` | Tab switcher with content panes |
| `Picker` | Fuzzy command palette |
| `StatusBar` | Left/right footer |
| `Help` | Auto-generated keybinding overlay |
| `DualPane` | Two-pane layout |
| `KeyBind` | Keybinding definition |
| `UpdateConfig` | Self-update configuration |
| `ToastMsg` | Toast notification message |
